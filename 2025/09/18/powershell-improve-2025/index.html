美化 PowerShell：Oh My Posh 全攻略 (2025)

發表於 2025-09-18 更新於 2025-09-19 分類於 windows Disqus：

這篇你將會獲得：

  1. 乾淨可重現的 Profile 初始化步驟
  2. 提升效率的補全與權限操作設定
  3. 一套漂亮又資訊量足夠的 Prompt 主題
  4. Windows Terminal 配色與字型建議

收尾檢查清單

  • Profile 是否已包含：PSReadLine、oh-my-posh、sudo 別名

  • 是否能看到彩色 Git 分支與圖示
  • 是否能正常用 sudo 執行管理員指令
